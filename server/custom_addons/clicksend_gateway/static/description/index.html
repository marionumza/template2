ClickSend SMS Gateway

This module is a supporting application for "SMS Notification" module. The module allows you to send small service notifications to your valuable customer's mobile numbers through ClickSend SMS Gateway. To use this module first you need to install "SMS Notification".

Basic Features
  • This application provides the required SMS gateway facility to "SMS Notification" application for sending an SMS notification to the mobile number.
  • ClickSend Gateway module is able to send text SMS directly to the customers, suppliers, buyers, any partners or any mobile number.
  • Send single or group SMS using ClickSend SMS gateway.
  • Track status of all sent SMS by ClickSend SMS gateway.
