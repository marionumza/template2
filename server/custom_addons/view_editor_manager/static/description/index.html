View Editor

Editing Views

To Edit view one has to click on top right corner of the navbar, After clicking it will open a popup with view names, click on any of the shown view to start editing.

Editing Form View

Form View is a very basic view one can edit, add, delete new fields and existing fields into it. The screen shot depicts the usage of form view, one can hover over any element and they can see 3 options.

1) Is for adding fields.
2) Is for modifying field properties.
3) Is for deleting fields.

Adding Fields

Once clicking on add button you will see a Dialog. This Dialog will determine what kind of field you want on your view.

On Dialog will see :-

1)Position:- Position determine where do you want your field to be appear.
2)Select Field :- This determines what kind of field you want e.g we can create new or use existing field.
3)Fields:- If chosen existing one will be able to see only fields available in the view If chosen new one can create new view field.

Editing Fields

One can easily edit any fields visible to them on the view. i.e edit name, widget, attrs etc.

Deleting Fields

One can easily delete any field if it has no reference to any other field.

Adding A Smart Button

Yes you heard it true, one can easily add a new smart button into the form view.

A model will pop up fill all fields:-

1) Button Position :- Position of the button to be placed it's inside only.
2) Button Label :- Name of the button.
3) Icons :- Icon to be used for the button.
4) Relation :- Relation field for smart button

Adding A Tab

You can easily add tabs to the form view

A model will pop up fill all fields:-

1)Position :- Position of the tab to be placed.
2)Name :- Name of the tab
3)String:- String of the tab.

Adding A View To Model

In View Editor you can create a new view from scratch. All you have to do is click on add new view.
