Marketing Automation Campaign

Marketing Configuration

Create Marketing Configuration by selection object and field. Our base object will be contact and field could be anything on which you wish to perform campaign. The Sub Configuration are domain or mapping for the object, They are created automatically when you use domain. It serves same purpose for creating marketing flow.

Create Marketing Campaign

Enter your campaign name and in model select configuration that you have created. Additionally you will see domain field which is generated from configuration that you selected. You can edit the domain to target your contacts.

Activity Builder

Clicking on Activity Builder button, A Canvas will be provided where you can define your marketing flow.

On left You will see tabs:-
1) Actions :- send email or server action.
2) Flow Controls :- Timer for controlling your flow.
3) Conditions :- Conditions or domain to target your audience.

Sending Email

Send Mail:- To send a mail, drag and drop Send Mail item on canvas. Provide 3 options for sending mail

A model will pop up fill all fields:-
1) Name :- Subject of your email.
2) Model :- Model Object of your email
3) template :- Mail template of your email.

Server Action

Server Action:- To run a server action, drag and drop Server Action item on canvas. Provide 3 options for sending mail

A model will pop up fill all fields:-
1) Name :- Subject of your email.
2) Model :- Model Object of your email
3) Server Action :- To run server action provide server action reference.

Flow Controls

Wait for some time :- The flow control plays role when you want to wait for a particular amount of time.

A model will pop up fill all fields:-
1) Wait for :- Amount of time in integer.
2) Waiting for :- Define minutes, hours etc.

Filtering Contacts

Domain :- We can always filter our contact and send email according to that.

A model will pop up fill all fields:-
1) Model :- Select model for which you are building domain.
2) Domain :- Condition to filter your domain.
3) Map field:- If your object is non contact object then you have to select field which has relation with contact.

Action Types

To complete our flow we all have to connect lines in canvas, It will allow us to define connection of activities.

Hover over welcome mail block, You will see 9 options also called wheel of fortune.

1) on send mail :- Perform associated action after sending email.
2) on email bounced :- Perform associated action after email bounced.
3) on child activity :- Perform associated action after child activity.
4) on email open :- Perform associated action after email open.
5) on email not open:- Perform associated action after email not open.
6) email replied :- Perform associated action after email replied.
7) email not replied :- Perform associated action after email not replied.
8) email clicked :- Perform associated action after email clicked.
9) email not clicked :- Perform associated action after email not clicked.

Completing Flow

It is necessary to connect all the lines from one activity to another, Without connection activity don't make any sense.

Exploring Backend

After completing backend you will see that there are few data inserted automatically. These Data are made up from Activity builder.

1) Rule:- This Rule is the condition we have provided in activity builder. 2) Mail Activities:- This are the lines of activities we build previously.

Generate Mail Members

After flow is ready we have to click on get started button and click on generate members button or a cron will run every 24 hours to generate members of the contact.

Generate Mail Activity

After generating mail members we will have member activity automatically generated.

Execute Activities

After Creating members and activities the mail activities must be executed by the cron every 1 hour.
