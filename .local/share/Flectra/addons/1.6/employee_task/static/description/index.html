The number of Tasks assigned to the Employee

Live Preview

To empower the user by providing the decision making knowledge is one of the key areas
to focus while implementing ERP.

Our effort towards providing analysed information mining through the already existed data
produces this another new Flectra module.

The main philosophy to develop this module is connecting project and human resource management
application to enable the right set of data while the user is accessing employee record in Flectra.

This module is specifically for the Flectra implementation which is using Human resource and
project management system. It helps to bring the power of project management data to Human
resource application.

In Flectra, project task is assigned to the user, not the employee. If you want to see the number of
tasks assigned to the employee on the employee form view and kanban view then this module is for you.

You can control showing such data to limited number of users as well.

After installing this module, you can check the following access right to allow the particular
user to see the number of tasks assigned to the employee in kanban and form view.

Following is the employee kanban view to display the number of tasks assigned to the employee.

Following is the form view of the employee. In the upper right side, Flectra displays the number of
tasks assigned to the employee.

By Clicking on that 'Tasks' button, Flectra will open the list of tasks group by stages assigned to this employee.
